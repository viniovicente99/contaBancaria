# 💰 Banco do Brazil com Z

Um projeto simples em Java que simula um sistema bancário via terminal.
A ideia aqui é praticar lógica de programação, estruturas de controle e organização básica de um sistema CRUD.

---

## 🚀 Funcionalidades

O sistema apresenta um menu interativo com as seguintes opções:

* Criar conta
* Listar todas as contas
* Buscar conta por número
* Atualizar dados da conta
* Apagar conta
* Sacar
* Depositar
* Transferir valores entre contas
* Sair do sistema

> ⚠️ Atualmente, as opções estão como estrutura inicial (prints no console). A lógica completa pode ser implementada aos poucos.

---

## 🛠️ Tecnologias usadas

* Java
* Terminal / Console
* `Scanner` para entrada de dados

---

## 📦 Como executar

1. Clone o repositório:

```bash
git clone https://github.com/viniovicente99/seu-repo.git
```

2. Acesse a pasta do projeto:

```bash
cd seu-repo
```

3. Compile o projeto:

```bash
javac conta_bancaria/Menu.java
```

4. Execute:

```bash
java conta_bancaria.Menu
```
